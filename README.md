# 🎓 IELTS Exam Management System

> 📅 Developed during **2022–2023**  
> 💻 Built with **C#**, designed using **Domain‑Driven Design (DDD)** principles  
> 🧩 Includes **Admin Panel** & **Front Panel** for complete exam management  

---

## 🧭 Overview

**IELTS Exam Management System** is a full-featured platform for managing and conducting IELTS exams.  
It provides all necessary tools for scheduling, administering, and evaluating the four IELTS skills — **Listening**, **Reading**, **Writing**, and **Speaking**.

Perfect for institutes and exam centers who want an organized and automated management experience.

---

## ⚙️ Key Features

✅ **Comprehensive IELTS Management**
- Full support for **four skills**:
  - 🎧 Listening  
  - 📖 Reading  
  - ✍️ Writing (including file upload for writing tasks)  
  - 🗣️ Speaking (with configurable interview time slots)

✅ **Question Bank & Exam Creator**
- Define and categorize **various question types** (multiple choice, text-based, essay, etc.)  
- Create and schedule new tests easily  
- Randomized exam generation and time management

✅ **Admin Panel**
- Manage candidates, exams, results, and interviews  
- Define exam time frames and monitor progress  
- Fully role-based access for secure management

✅ **Front Panel**
- Register for exams  
- Upload writing section responses  
- View schedule, results, and score breakdown

✅ **Architecture**
- Designed with **DDD** for clarity and scalability  
- Developed in **C# / .NET Framework**
- Modular structure for future extension
